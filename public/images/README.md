# Carpeta de Imágenes

Aquí puedes añadir tus fotos para que se vean en la web.

### Instrucciones:
1. Mete tus fotos en las carpetas correspondientes:
   - `portfolio/`: Para las fotos de la sección Fotografía.
   - `commercial/`: Para la sección Comercial.
   - `hero/`: Para las fotos grandes de fondo (como la de la página de inicio).

2. **Cómo usarlas en el código:**
   Si guardas una foto llamada `foto1.jpg` en `public/images/portfolio/`, en el código de React la ruta será:
   `"/images/portfolio/foto1.jpg"`

### Recomendación:
- Usa formato `.webp` si puedes para que carguen más rápido.
- No subas fotos de más de 2MB.
